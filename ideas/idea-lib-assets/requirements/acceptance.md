# Acceptance Criteria

## Definition of done
- Library supports image, SVG, and code snippet assets
- Assets can be organized by scope and tagged
- A code snippet can be rendered to PNG for event page use

## Acceptance tests
- Import a React component (e.g., `components/internal/email/templates/WelcomeEmail.tsx`) as a library item
- Render and export the component to PNG
- Search by asset type and scope returns the item
