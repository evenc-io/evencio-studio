# Functional Requirements

## Must have
- Library supports asset types: image, SVG, code snippet (React/HTML)
- Assets can be organized by scope: global, org, event, personal
- Tagging, search, and filtering by type and scope
- Collections and favorites
- Export code snippets to PNG for event pages

## Should have
- Versioning for reusable assets
- Presets/variants for different sizes
- Basic governance: draft vs published

## Nice to have
- Smart views (most used, recently updated)
- Usage analytics
