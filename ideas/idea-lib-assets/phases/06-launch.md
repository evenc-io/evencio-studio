# Launch

## Rollout plan
- Internal alpha with OSS contributors + 1–2 partner orgs
- Document import + snippet registration workflows
- Gradual enablement of code snippet assets (feature flag)

## Communication plan
- Changelog entry and docs update
- Short guide: “Create your first library asset”
- Demo video/gif of snippet → PNG export

## Monitoring
- Track search latency and render time
- Error rates for snippet render/export
- Asset creation and reuse metrics

## Rollback plan
- Disable snippet rendering feature flag
- Revert to image/SVG-only library
